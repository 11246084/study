## Unit 1｜print 輸出

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `print()` 在螢幕上顯示文字和數字
- 了解 `print()` 的基本格式
- 知道為什麼文字要加引號

---

### 📖 概念說明

#### 什麼是 print()？

想像你在跟電腦說話，你對它說：「把這個顯示在螢幕上！」  
`print()` 就是你對電腦下的這個指令。

**基本格式：**

```python
print("你想顯示的文字")
```

⚠️ **注意**：文字一定要用**引號**（`" "` 或 `' '`）包起來，不然電腦會以為是變數名稱而報錯。

#### 執行流程

```
你寫 print("Hello")
        ↓
Python 讀到這一行
        ↓
螢幕顯示：Hello
```

---

### 💻 範例程式（含逐行說明）

```python
# 檔案：ch4_1_level1.py

print("你好，世界！")        # 顯示一段文字 → 輸出：你好，世界！
print(10 + 20)              # 計算後顯示結果 → 輸出：30
print("答案是", 10 + 20)    # 同時顯示文字和數字 → 輸出：答案是 30
```

**執行結果：**
```
你好，世界！
30
答案是 30
```

> 💡 每個 `print()` 執行完，游標會自動跳到下一行，所以多個 `print` 的結果會分開顯示。

---

### 📌 print() 可以顯示什麼？

| 類型 | 範例 | 輸出 |
|------|------|------|
| 文字（字串）| `print("蘋果")` | 蘋果 |
| 整數 | `print(100)` | 100 |
| 計算式 | `print(5 * 3)` | 15 |
| 文字＋數字 | `print("結果是", 5*3)` | 結果是 15 |

## Unit 2｜input 輸入與運算

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `input()` 接收使用者從鍵盤輸入的資料
- 理解 `input()` 回傳的是「字串」
- 使用 `int()` 和 `float()` 將字串轉成數字，進行運算

---

### 📖 概念說明

#### 什麼是 input()？

`input()` 讓程式「暫停」，等待使用者在鍵盤輸入文字，按下 Enter 後程式才繼續。

```python
變數 = input("請輸入提示文字：")
```

#### ⚠️ 重要：input() 回傳的是「字串」

不管你輸入什麼，`input()` 都把它當成文字（字串），不是數字！

```
使用者輸入 → "20"（字串，不是數字 20）
```

#### 型別轉換流程

```
input() 得到 "20"（字串）
        ↓  int("20")
        20（整數，可以做加減乘除）
```

**轉換方式：**

| 函式 | 用途 | 範例 |
|------|------|------|
| `int()` | 字串轉整數 | `int("20")` → `20` |
| `float()` | 字串轉浮點數 | `float("3.14")` → `3.14` |

---

### 💻 範例程式（含逐行說明）

```python
# 檔案：ch4_2_level1.py

name = input("請輸入你的名字：")    # 等待輸入，存入 name 變數（字串）
print("你好，", name)               # 顯示歡迎訊息

age = input("請輸入你的年齡：")     # 輸入年齡（此時是字串！）
age = int(age)                      # 轉換成整數，才能做計算
print("明年你", age + 1, "歲")      # 加 1 後顯示
```

**執行過程：**
```
請輸入你的名字：小明
你好， 小明
請輸入你的年齡：18
明年你 19 歲
```

## Unit 3｜if 條件判斷

### 🎯 學習目標

完成本單元後，你能夠：
- 理解「條件判斷」的概念
- 使用 `if` 和 `if...else` 語法
- 知道縮排（indent）的重要性

---

### 📖 概念說明

#### 什麼是條件判斷？

生活中很多決定都是「如果...就...」的邏輯：
- 如果天氣好，就去公園
- 如果成績及格，就放暑假

Python 的 `if` 就是幫你寫出這種邏輯。

#### 基本語法

```python
if 條件：
    條件成立時執行的程式碼   # 注意：這裡要縮排（按 Tab 或 4 個空格）
```

#### if...else 語法

```python
if 條件：
    條件成立時執行
else：
    條件不成立時執行
```

#### 執行流程

```
程式到達 if
        ↓
  判斷條件是 True 還是 False？
    ↙                    ↘
  True                 False
執行 if 區塊          執行 else 區塊
        ↓                    ↓
      繼續往下執行
```

#### 關係運算子（用來寫條件）

| 運算子 | 意義 | 範例 | 結果 |
|-------|------|------|------|
| `==` | 等於 | `5 == 5` | True |
| `!=` | 不等於 | `5 != 3` | True |
| `>` | 大於 | `10 > 5` | True |
| `<` | 小於 | `3 < 1` | False |
| `>=` | 大於等於 | `5 >= 5` | True |
| `<=` | 小於等於 | `3 <= 5` | True |

---

### 💻 範例程式（含逐行說明）

```python
# 檔案：ch5_1_level1.py

age = int(input("請輸入年齡："))   # 輸入年齡並轉成整數

if age >= 18:                       # 如果年齡大於等於 18
    print("你已成年！")             # 條件成立，顯示此訊息
else:                               # 否則（年齡小於 18）
    print("你尚未成年。")           # 條件不成立，顯示此訊息

print("程式結束")                   # 不管條件如何，都會執行
```

**執行結果（輸入 20）：**
```
請輸入年齡：20
你已成年！
程式結束
```

**執行結果（輸入 15）：**
```
請輸入年齡：15
你尚未成年。
程式結束
```

## Unit 4｜多條件 if

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `if...elif...else` 處理多種情況
- 理解程式只會執行第一個符合條件的區塊
- 設計成績等第、BMI 等多段判斷程式

---

### 📖 概念說明

#### 什麼是 elif？

當有三種以上的情況需要判斷時，`else if`（縮寫為 `elif`）可以幫你新增更多條件分支。

```python
if 條件一：
    執行區塊一
elif 條件二：
    執行區塊二
elif 條件三：
    執行區塊三
else：
    以上都不符合時執行
```

#### 執行流程（以成績分級為例）

```
輸入分數
    ↓
分數 >= 90？
  ↙ Yes → 印出 "A"
  ↓ No
分數 >= 80？
  ↙ Yes → 印出 "B"
  ↓ No
分數 >= 70？
  ↙ Yes → 印出 "C"
  ↓ No
       → 印出 "F"
```

> ⚠️ 程式從上往下檢查，**找到第一個符合的條件就執行，後面不再檢查**。

---

### 💻 範例程式（含逐行說明）

```python
# 檔案：ch5_2_level1.py

score = int(input("請輸入分數（0~100）："))   # 輸入分數

if score >= 90:                               # 如果分數 >= 90
    print("等第：A")
elif score >= 80:                             # 否則如果 >= 80
    print("等第：B")
elif score >= 70:                             # 否則如果 >= 70
    print("等第：C")
elif score >= 60:                             # 否則如果 >= 60
    print("等第：D")
else:                                         # 以上都不符合（< 60）
    print("等第：F")
```

**執行結果（輸入 85）：**
```
請輸入分數（0~100）：85
等第：B
```

## Unit 5｜list 串列

### 🎯 學習目標

完成本單元後，你能夠：
- 理解串列是什麼，並建立一個串列
- 用索引讀取串列中的元素
- 新增（append）和刪除（remove）串列元素

---

### 📖 概念說明

#### 什麼是串列？

想像一排有編號的盒子，每個盒子可以放一個東西：

```
索引：  0       1       2       3
     ┌──────┬──────┬──────┬──────┐
     │"蘋果"│"香蕉"│"西瓜"│"葡萄"│
     └──────┴──────┴──────┴──────┘
```

這排盒子就是「串列」，Python 用方括號 `[]` 來表示。

#### 建立串列

```python
水果 = ["蘋果", "香蕉", "西瓜", "葡萄"]
```

#### 讀取元素（用索引）

```python
print(水果[0])   # 蘋果（第一個，索引從 0 開始！）
print(水果[1])   # 香蕉
print(水果[-1])  # 葡萄（-1 代表最後一個）
```

> ⚠️ **索引從 0 開始**，不是從 1 開始！這是初學者最常搞錯的地方。

---

### 💻 範例程式（含逐行說明）

```python
# 檔案：ch6_1_level1.py

fruits = ["蘋果", "香蕉", "西瓜"]     # 建立串列，含三個元素

print(fruits)                          # 印出整個串列
print(fruits[0])                       # 印出索引 0 的元素：蘋果
print(fruits[2])                       # 印出索引 2 的元素：西瓜
print("串列長度：", len(fruits))       # len() 回傳元素數量：3

fruits.append("葡萄")                  # 在末尾新增「葡萄」
print(fruits)                          # ['蘋果', '香蕉', '西瓜', '葡萄']

fruits.remove("香蕉")                  # 刪除「香蕉」
print(fruits)                          # ['蘋果', '西瓜', '葡萄']
```

**執行結果：**
```
['蘋果', '香蕉', '西瓜']
蘋果
西瓜
串列長度： 3
['蘋果', '香蕉', '西瓜', '葡萄']
['蘋果', '西瓜', '葡萄']
```

## Unit 6｜range 數列

### 🎯 學習目標

完成本單元後，你能夠：
- 理解 `range()` 會產生一連串的數字
- 使用 `range(n)`、`range(start, stop)`、`range(start, stop, step)`

---

### 📖 概念說明

#### 什麼是 range()？

`range()` 就像一個「數字產生器」，可以產生一連串的整數。

#### 三種用法

**只有一個參數：**
```python
range(5)   # 產生：0, 1, 2, 3, 4（不含 5！）
```

視覺化：
```
┌─┬─┬─┬─┬─┐
│0│1│2│3│4│
└─┴─┴─┴─┴─┘
```

**兩個參數（start, stop）：**
```python
range(2, 6)   # 產生：2, 3, 4, 5（不含 6！）
```

**三個參數（start, stop, step）：**
```python
range(0, 10, 2)   # 產生：0, 2, 4, 6, 8（每次加 2）
range(10, 0, -1)  # 產生：10, 9, 8, ..., 1（倒數）
```

> ⚠️ `range()` **不包含** stop 這個數字，只到 stop - 1 為止！

---

### 💻 範例程式（含逐行說明）

```python
# 檔案：ch7_1_level1.py

# 用法一：range(n)
print(list(range(5)))          # [0, 1, 2, 3, 4]

# 用法二：range(start, stop)
print(list(range(1, 6)))       # [1, 2, 3, 4, 5]

# 用法三：range(start, stop, step)
print(list(range(0, 11, 2)))   # [0, 2, 4, 6, 8, 10]
print(list(range(5, 0, -1)))   # [5, 4, 3, 2, 1]
```

## Unit 7｜for 迴圈

### 🎯 學習目標

- 使用 `for` 迴圈搭配串列和 `range()`

---

### 📖 概念說明

#### 什麼是 for 迴圈？

想像你要逐一叫出班上每位同學的名字，你會從第一個叫到最後一個。  
`for` 迴圈就是幫你「逐一處理」每個東西。

```python
for 變數 in 可迭代物件:
    對每個元素執行的程式碼
```

#### 執行流程（以串列為例）

```
students = ["Alice", "Bob", "Carol"]
for name in students:
    print(name)
```

```
第一次：name = "Alice" → 印出 Alice
第二次：name = "Bob"   → 印出 Bob
第三次：name = "Carol" → 印出 Carol
→ 串列結束，迴圈停止
```

#### for 搭配 range()

```python
for i in range(5):
    print(i)
```

```
第一次：i = 0 → 印出 0
第二次：i = 1 → 印出 1
...
第五次：i = 4 → 印出 4
→ range 結束，迴圈停止
```

## Unit 8｜迴圈應用

### 🎯 學習目標

完成本單元後，你能夠：
- 理解 `while` 迴圈的概念與語法
- 使用 `while` 搭配計數器和累加器
- 知道無限迴圈如何發生以及如何避免

---

### 📖 概念說明

#### 什麼是 while 迴圈？

`for` 迴圈是「跑固定次數」，`while` 迴圈是「只要條件成立就一直跑」。

```python
while 條件:
    條件成立時執行的程式碼
```

#### 執行流程

```
設定初始條件
        ↓
  判斷條件是 True 嗎？
    ↙ Yes              ↘ No
  執行程式碼         離開迴圈，往下繼續
    ↓
  更新條件（很重要！）
    ↓
  回到判斷條件
```

#### 累加器模式

「累加器」是一個變數，每次迴圈都把新的值加進去：

```python
total = 0            # 累加器，初始為 0
i = 1                # 計數器，初始為 1
while i <= 5:
    total = total + i   # 把 i 加進 total
    i = i + 1           # 計數器 +1，讓迴圈最終停止
print(total)         # 15
```

---

### 💻 範例程式（含逐行說明）

```python
# 檔案：ch7_3_level1.py

# 範例：計算 1 加到 n

n = int(input("請輸入 n："))   # 輸入上限值
total = 0                       # 累加器，初始值為 0
i = 1                           # 計數器，從 1 開始

while i <= n:                   # 當 i 還沒超過 n，繼續執行
    total = total + i           # 把目前的 i 加進 total
    i = i + 1                   # i 加 1，往下一個數字前進

print("1 加到", n, "的總和是：", total)
```

**執行結果（輸入 5）：**
```
請輸入 n：5
1 加到 5 的總和是： 15
```

下面的程式有一個嚴重問題，是什麼？

count = 0
while count < 5:

<details>
<summary>✅ 答案</summary>

</details>

### ⚠️ 常見錯誤說明

| 錯誤類型 | 說明 | 修正方式 |
|---------|------|---------|
| 無限迴圈 | while 條件永遠是 True | 確保迴圈內有更新條件的敘述 |
| 計數器未初始化 | 使用前沒有設定初始值 | 在 while 前先設定 `i = 0` 或 `i = 1` |
| 縮排錯誤 | while 區塊內的程式沒縮排 | 按 Tab 鍵縮排 |
