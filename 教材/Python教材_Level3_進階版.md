## Unit 1｜print 輸出

### 📖 概念摘要

`print()` 的 `sep` 和 `end` 參數控制分隔與結尾字元。格式化輸出建議優先使用 f-string：

```python
f"{value:{fill}{align}{width}.{precision}{type}}"
```

對齊符：`<` 靠左、`>` 靠右、`^` 置中；`+` 顯示正負號；填充字元放在對齊符之前。

```python
f"{'text':*^20}"    # ***********text***********（填充 *，置中，寬度 20）
f"{3.14159:.3f}"    # 3.142
f"{42:08b}"         # 00101010（二進位，補零，寬度 8）
```

## Unit 2｜input 輸入與運算

### 📖 概念摘要

`input()` 回傳字串，型別轉換錯誤會拋出 `ValueError`，可用 `try/except` 處理。  
`eval()` 可解析任意 Python 運算式，但存在安全風險（不應用於不信任的輸入）。  
一次輸入多個值可用 `split()` 搭配解包：

```python
a, b, c = map(int, input().split())
```

`map(func, iterable)` 對序列中每個元素套用函式，回傳 map 物件。

## Unit 3｜if 條件判斷

### 📖 概念摘要

Python 支援連鎖比較（`0 < x < 10`）和三元運算式（`x if 條件 else y`）。  
邏輯運算子 `and`、`or`、`not` 遵循短路求值：

- `A and B`：A 為 False 時不評估 B
- `A or B`：A 為 True 時不評估 B

```python
# PEP 8 建議直接用布林值
if is_valid:          # 不要寫 if is_valid == True
if not is_empty:      # 不要寫 if is_empty == False
```

`in` 運算子可快速判斷成員：`if x in [1, 2, 3]` 比三個 `or` 更易讀。

## Unit 4｜多條件 if

### 📖 概念摘要

`if/elif/else` 鏈的效率：條件從上往下依序判斷，找到符合的即停止。  
把**最常發生的條件放在最前面**可提升效能。  
複雜的多條件邏輯可考慮用字典替代 if/elif：

```python
# 傳統寫法
if day == "Monday":
    plan = "會議"
elif day == "Tuesday":
    plan = "程式設計"

# 字典查表（更簡潔）
schedule = {
    "Monday": "會議",
    "Tuesday": "程式設計",
    "Wednesday": "設計審查",
}
plan = schedule.get(day, "自由時間")  # 找不到 key 時回傳預設值
```

## Unit 5｜list 串列

### 📖 概念摘要

串列支援切片（`[start:end:step]`）、`sorted()` vs `sort()` 的差異（前者回傳新串列，後者原地修改）。  
串列生成式（List Comprehension）：

```python
squares = [x**2 for x in range(1, 11)]
evens = [x for x in range(20) if x % 2 == 0]
flat = [x for row in matrix for x in row]    # 展開二維串列
```

`sorted()` 的 `key` 參數：

```python
students = [("Alice", 92), ("Bob", 75), ("Carol", 88)]
sorted(students, key=lambda x: x[1], reverse=True)  # 依分數降冪
```

二維串列轉置：

```python
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
transpose = [[row[i] for row in matrix] for i in range(3)]
```

## Unit 6｜range 數列

### 📖 概念摘要

`range` 是惰性序列（lazy sequence），不會預先產生所有數字，節省記憶體。  
常見進階模式：

```python
# 反向走訪索引
for i in range(len(lst) - 1, -1, -1):
    ...

# 與 zip 結合同時走訪兩個範圍
for a, b in zip(range(0, 5), range(5, 10)):
    print(a, b)    # (0,5), (1,6), (2,7), (3,8), (4,9)
```

`range` 物件支援 `in` 判斷（O(1) 效能，不走訪整個序列）：

```python
if 50 in range(0, 101, 2):    # 常數時間
    print("是偶數且在範圍內")
```

## Unit 7｜for 迴圈

### 📖 概念摘要

`for` 迴圈可走訪任何可迭代物件（iterable）。  
進階工具：

```python
enumerate(iterable, start=0)    # 回傳 (index, value) 配對
zip(*iterables)                 # 平行走訪多個序列（以最短者為準）
```

迴圈中的 `else` 子句：

```python
for item in collection:
    if condition(item):
        break
else:
    # 只有在迴圈沒有被 break 中斷時才執行
    print("未找到符合條件的元素")
```

串列生成式等效於 for 迴圈，但通常更快（CPython 層級優化）。

## Unit 8｜迴圈應用

### 📖 概念摘要

`while` 迴圈適用於次數不確定、依條件驅動的重複。  
`break` 跳出最內層迴圈；`continue` 跳過本次迭代；巢狀迴圈的 `break` 只跳出一層。  

常見模式：

```python
# 哨兵值模式（輸入 q 結束）
while (line := input()) != "q":
    process(line)

# 重試模式
for attempt in range(3):
    result = try_operation()
    if result:
        break
else:
    print("三次嘗試均失敗")

# 雙指標（兩端同時走訪）
left, right = 0, len(lst) - 1
while left < right:
    lst[left], lst[right] = lst[right], lst[left]
    left += 1; right -= 1
```

`:=` 是海象運算子（Python 3.8+），可在運算式中賦值。
