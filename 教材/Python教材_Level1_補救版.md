## Unit 1｜print 輸出

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `print()` 在螢幕上顯示文字和數字
- 知道文字為什麼要加引號
- 用 `print()` 同時顯示多個東西

---

### 📖 概念說明

#### 什麼是 print()？

想像你在跟電腦說話，你對它說：「把這個顯示在螢幕上！」  
`print()` 就是你對電腦下的這個指令。

```python
print("你想顯示的文字")
```

⚠️ **注意**：文字一定要用**引號**（`" "` 或 `' '`）包起來，不然電腦會以為是變數名稱而報錯。

#### print() 執行流程

```
你寫 print("Hello")
        ↓
Python 讀到這一行
        ↓
螢幕顯示：Hello
```

每個 `print()` 執行完，游標自動跳到**下一行**。

---

### 💻 完整範例（含逐行說明）

```python
print("你好，世界！")        # 顯示一段文字 → 輸出：你好，世界！
print(10 + 20)              # 計算後顯示結果 → 輸出：30
print("答案是", 10 + 20)    # 同時顯示文字和數字 → 輸出：答案是 30
```

**執行結果：**
```
你好，世界！
30
答案是 30
```

> 💡 用逗號連接文字和數字時，`print()` 會自動在中間加一個空格。

---

### 📌 print() 可以顯示什麼？

| 類型 | 範例 | 輸出 |
|------|------|------|
| 文字（字串）| `print("蘋果")` | 蘋果 |
| 整數 | `print(100)` | 100 |
| 計算式 | `print(5 * 3)` | 15 |
| 文字＋數字 | `print("結果是", 5*3)` | 結果是 15 |

## Unit 2｜input 輸入與運算

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `input()` 接收使用者從鍵盤輸入的資料
- 理解 `input()` 回傳的是「字串」，不是數字
- 使用 `int()` 和 `float()` 將字串轉成數字，進行運算

---

### 📖 概念說明

#### 什麼是 input()？

`input()` 讓程式**暫停**，等待使用者在鍵盤輸入文字，按下 Enter 後程式才繼續。

```python
變數 = input("請輸入提示文字：")
```

#### ⚠️ 超重要：input() 回傳的是「字串」！

不管你輸入什麼，`input()` 都把它當成**文字（字串）**。

```
使用者輸入 20 → 程式得到的是 "20"（字串），不是數字 20
```

這就像是你寫了一張寫著「20」的紙條給電腦，但紙條上的 20 不是真正的數字，不能直接拿來加減。

#### 型別轉換

要把「字串」變成「數字」，需要用轉換函式：

| 函式 | 用途 | 範例 |
|------|------|------|
| `int()` | 字串轉**整數** | `int("20")` → `20` |
| `float()` | 字串轉**浮點數（小數）** | `float("3.14")` → `3.14` |

#### 完整流程

```
input() 得到 "20"（字串）
        ↓
   int("20")
        ↓
   20（整數，可以做加減乘除）
```

---

### 💻 完整範例（含逐行說明）

```python
name = input("請輸入你的名字：")    # 等待輸入，存入 name（字串）
print("你好，", name)               # 顯示歡迎訊息

age = input("請輸入你的年齡：")     # 輸入年齡（此時是字串！）
age = int(age)                      # 轉換成整數，才能做計算
print("明年你", age + 1, "歲")      # 加 1 後顯示
```

**執行過程：**
```
請輸入你的名字：小明
你好， 小明
請輸入你的年齡：18
明年你 19 歲
```

## Unit 3｜if 條件判斷

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `if...else` 讓程式做二擇一的決定
- 知道關係運算子（`>`、`<`、`==` 等）的用法
- 理解縮排的重要性

---

### 📖 概念說明

#### 什麼是條件判斷？

生活中的邏輯：
- **如果**天氣好，**就**去公園
- **如果**成績及格，**就**放假，**否則**補考

Python 的 `if` 就是幫你寫出這種邏輯：

```python
if 條件:
    條件成立時執行    # ← 要縮排（4 個空格）
else:
    條件不成立時執行  # ← 要縮排（4 個空格）
```

#### 執行流程

```
程式到達 if
        ↓
  條件是 True 還是 False？
    ↙ True              ↘ False
執行 if 區塊         執行 else 區塊
        ↓                    ↓
      繼續往下執行
```

#### 關係運算子（用來寫條件）

| 運算子 | 意義 | 範例 | 結果 |
|-------|------|------|------|
| `==` | 等於 | `5 == 5` | True |
| `!=` | 不等於 | `5 != 3` | True |
| `>` | 大於 | `10 > 5` | True |
| `<` | 小於 | `3 < 1` | False |
| `>=` | 大於等於 | `5 >= 5` | True |
| `<=` | 小於等於 | `3 <= 5` | True |

---

### 💻 完整範例（含逐行說明）

```python
age = int(input("請輸入年齡："))   # 輸入年齡並轉成整數

if age >= 18:                       # 如果年齡大於等於 18
    print("你已成年！")             # 條件成立，顯示此訊息
else:                               # 否則（年齡小於 18）
    print("你尚未成年。")           # 條件不成立，顯示此訊息

print("程式結束")                   # 不管條件如何，都會執行
```

**輸入 20：** `你已成年！` → `程式結束`  
**輸入 15：** `你尚未成年。` → `程式結束`

## Unit 4｜多條件 if

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `if...elif...else` 處理三種以上的情況
- 理解程式只會執行**第一個**符合條件的區塊
- 設計成績等第判斷程式

---

### 📖 概念說明

#### 什麼是 elif？

當有三種以上的情況需要判斷時，加入 `elif`（else if 的縮寫）：

```python
if 條件一:
    執行區塊一
elif 條件二:
    執行區塊二
elif 條件三:
    執行區塊三
else:
    以上都不符合時執行
```

#### 執行流程（成績分級為例）

```
輸入分數
    ↓
分數 >= 90？  → Yes → 印出 "優等"
    ↓ No
分數 >= 80？  → Yes → 印出 "良好"
    ↓ No
分數 >= 70？  → Yes → 印出 "普通"
    ↓ No
              → 印出 "需加強"
```

> ⚠️ 程式從上往下檢查，**找到第一個符合的條件就執行，後面不再看**。

---

### 💻 完整範例（含逐行說明）

```python
score = int(input("請輸入分數（0~100）："))  # 輸入分數

if score >= 90:           # 90 分以上
    print("優等")
elif score >= 80:         # 80~89 分（不用寫 score < 90，因為走到這裡已經 < 90）
    print("良好")
elif score >= 70:         # 70~79 分
    print("普通")
elif score >= 60:         # 60~69 分
    print("及格")
else:                     # 60 分以下
    print("需加強")
```

**輸入 85：** `良好`  
**輸入 55：** `需加強`

## Unit 5｜list 串列

### 🎯 學習目標

完成本單元後，你能夠：
- 建立串列（list）來儲存多個資料
- 用索引存取串列中的元素
- 使用 `append()` 新增元素，`len()` 計算長度

---

### 📖 概念說明

#### 什麼是串列？

串列就像是一排**有編號的格子**，每個格子可以存放一個資料：

```
  格子編號（索引）：  0      1      2      3
  格子裡的內容：    "蘋果" "香蕉" "橘子" "葡萄"
```

```python
fruits = ["蘋果", "香蕉", "橘子", "葡萄"]
```

#### 建立與存取

```python
fruits = ["蘋果", "香蕉", "橘子"]  # 建立串列

print(fruits[0])    # 輸出：蘋果（第一個，索引 0）
print(fruits[1])    # 輸出：香蕉（第二個，索引 1）
print(fruits[-1])   # 輸出：橘子（最後一個，用 -1）
```

> ⚠️ 索引從 **0** 開始，不是從 1！

#### 常用操作

| 操作 | 語法 | 說明 |
|------|------|------|
| 取得長度 | `len(fruits)` | 回傳元素數量 |
| 新增到末尾 | `fruits.append("葡萄")` | 加入新元素 |
| 修改元素 | `fruits[0] = "西瓜"` | 修改索引 0 的值 |

---

### 💻 完整範例（含逐行說明）

```python
scores = [85, 92, 78, 65, 90]  # 建立一個儲存分數的串列

print("第一個分數：", scores[0])    # 輸出：85
print("最後一個分數：", scores[-1]) # 輸出：90
print("共有", len(scores), "個分數") # 輸出：共有 5 個分數

scores.append(88)                   # 新增一個分數到末尾
print("新增後的串列：", scores)      # 輸出：[85, 92, 78, 65, 90, 88]

print("最高分：", max(scores))      # 輸出：92
print("最低分：", min(scores))      # 輸出：65
print("總分：", sum(scores))        # 輸出：498
```

| 錯誤 | 說明 |
|------|------|

## Unit 6｜range 數列

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `range()` 產生一串連續的數字
- 理解 `range(start, stop, step)` 三個參數的意義
- 搭配 `list()` 把 range 轉成看得見的串列

---

### 📖 概念說明

#### 什麼是 range()？

`range()` 就像是「自動幫你數數字的工具」，不用你一個一個輸入。

```python
range(5)          # 產生 0, 1, 2, 3, 4（從 0 到 4，共 5 個）
range(1, 6)       # 產生 1, 2, 3, 4, 5（從 1 到 5）
range(0, 10, 2)   # 產生 0, 2, 4, 6, 8（每次跳 2）
```

> ⚠️ `range(1, 6)` 包含 1，但**不包含 6**（到 5 為止）。

#### 三種用法

| 用法 | 說明 | 產生的數字 |
|------|------|-----------|
| `range(n)` | 從 0 到 n-1 | `range(5)` → 0,1,2,3,4 |
| `range(a, b)` | 從 a 到 b-1 | `range(1,4)` → 1,2,3 |
| `range(a, b, step)` | 從 a 到 b-1，每次跳 step | `range(0,10,3)` → 0,3,6,9 |

#### 轉成串列看看

```python
print(list(range(5)))          # [0, 1, 2, 3, 4]
print(list(range(1, 6)))       # [1, 2, 3, 4, 5]
print(list(range(0, 10, 2)))   # [0, 2, 4, 6, 8]
```

---

### 💻 完整範例（含逐行說明）

```python
# 顯示 1 到 5
nums = list(range(1, 6))          # 建立 [1, 2, 3, 4, 5]
print(nums)                        # 輸出：[1, 2, 3, 4, 5]

# 顯示 10 到 50，每次跳 10
tens = list(range(10, 51, 10))    # 建立 [10, 20, 30, 40, 50]
print(tens)                        # 輸出：[10, 20, 30, 40, 50]

# 計算 1 到 100 的總和
total = sum(range(1, 101))        # range(1, 101) 包含 1 到 100
print("總和：", total)            # 輸出：總和：5050
```

## Unit 7｜for 迴圈

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `for` 迴圈重複執行程式碼
- 搭配 `range()` 做固定次數的迴圈
- 走訪串列的每一個元素

---

### 📖 概念說明

#### 什麼是 for 迴圈？

想像你要把 5 個人的名字都叫一次：
- 手動版：`print("小明")`, `print("小華")`, `print("小麗")`, ...
- 迴圈版：讓程式自己重複執行！

```python
for 變數 in 序列:
    重複執行的程式碼    # 縮排！
```

#### 執行流程

```
for 迴圈
    ↓
從序列取出第一個值，存進變數
    ↓
執行縮排的程式碼
    ↓
還有下一個值嗎？
    ↙ Yes → 取出下一個值，重複執行
    ↓ No → 迴圈結束，繼續往下
```

---

### 💻 完整範例（含逐行說明）

**範例 1：走訪串列**
```python
fruits = ["蘋果", "香蕉", "橘子"]  # 準備一個串列

for fruit in fruits:                # 每次從 fruits 取出一個放進 fruit
    print(fruit)                    # 印出 fruit 的值
```
**輸出：**
```
蘋果
香蕉
橘子
```

**範例 2：搭配 range()**
```python
for i in range(1, 6):    # i 依序是 1, 2, 3, 4, 5
    print(i, "* 2 =", i * 2)  # 印出乘法表
```
**輸出：**
```
1 * 2 = 2
2 * 2 = 4
3 * 2 = 6
4 * 2 = 8
5 * 2 = 10
```

## Unit 8｜迴圈應用

### 🎯 學習目標

完成本單元後，你能夠：
- 使用 `while` 迴圈在「條件成立」時持續執行
- 理解 `break` 可以提前離開迴圈
- 設計簡單的計數與累加程式

---

### 📖 概念說明

#### 什麼是 while 迴圈？

`for` 迴圈是「做固定次數」，`while` 迴圈是「**只要條件成立就繼續做**」。

```python
while 條件:
    重複執行的程式碼    # 縮排！
```

#### 執行流程

```
while 迴圈
    ↓
  條件成立？
  ↙ Yes → 執行縮排的程式碼 → 回去判斷條件
  ↓ No → 結束迴圈，往下執行
```

> ⚠️ 如果條件**永遠成立**，就是「無窮迴圈」，程式永遠跑不完！記得在迴圈裡更新條件。

#### break 提前離開

```python
while True:           # 永遠成立（無窮迴圈）
    x = int(input("輸入正數："))
    if x > 0:
        break         # 輸入正數就跳出迴圈
print("你輸入了：", x)
```

---

### 💻 完整範例（含逐行說明）

**範例 1：計數迴圈**
```python
count = 1             # 從 1 開始計數
while count <= 5:     # 只要 count 不超過 5，就繼續
    print(count)      # 印出目前的 count
    count = count + 1 # count 加 1（如果忘記這行，會無窮迴圈！）
```
**輸出：** 1, 2, 3, 4, 5

**範例 2：累加到指定值**
```python
total = 0             # 初始總和
n = 1                 # 從 1 開始
while n <= 10:        # 計算 1+2+...+10
    total += n        # total = total + n
    n += 1            # n 加 1
print("1 到 10 的總和：", total)  # 輸出：55
```
